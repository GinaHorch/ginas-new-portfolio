## Overview

**DDDiamond Smash** is an interactive browser-based game developed for **DDD Perth** with starter code from She Codes Australia. The game challenges players to smash diamonds while avoiding decoys, incorporating **custom animations, sound effects, and a real-time scoring system** to enhance engagement.

## Key Features

- **Interactive Gameplay**: Players must smash diamonds while avoiding decoys, adding strategy and engagement.
- **Custom Animations & Sound Effects**: Dynamic visual and audio feedback for an immersive experience.
- **Real-Time Scoring**: Tracks high scores and live progress.
- **Responsive Design**: Ensures seamless performance across devices.
- **Optimised for Performance**: Developed with JavaScript best practices to maintain smooth interactions.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Development Tools**: Git, CodePen, Visual Studio Code

## Challenges and Learnings

- **Balancing Difficulty and Playability**: Ensuring the game is engaging but not overly challenging.
- **Performance Optimisation**: Managing animations and event listeners to avoid lag.
- **Cross-Browser Compatibility**: Testing across multiple browsers to ensure a consistent experience.

## Outcome

The **DDDiamond Smash Game** successfully engaged players at **DDD Perth**, demonstrating proficiency in **game development with JavaScript and CSS**. This project expanded my expertise in **interactive web development**, animations, and real-time game logic.
