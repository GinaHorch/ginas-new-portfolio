## Overview

This **Portfolio Website** was my first She Codes Plus project and served as a central hub for showcasing my **web development projects**, **technical skills**, and **Agile journey**. It is built with **modern front-end technologies**, ensuring **responsive design** and **optimised performance**. 

Visitors can explore my **tech projects**, **certifications**, and **background in Agile leadership**, reflecting my expertise in **web development, Scrum methodologies, and digital solutions**.

## Key Features

- **Responsive Design**: Fully optimised for desktop and mobile viewing.
- **Tech Projects Showcase**: Displays key web development and data engineering projects with interactive details.
- **Agile Journey Section**: Highlights my transition from law enforcement to tech, emphasising my **Scrum Master expertise**.
- **Dark Mode Support**: Enhances accessibility with theme adaptation.
- **Contact Form**: Integrated contact form for professional inquiries.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Development Tools**: Git, GitHub, Visual Studio Code
- **Deployment**: GitHub Pages

## Challenges and Learnings

- **Ensuring Accessibility**: Implemented semantic HTML and proper color contrast for better **UX**.
- **Mobile-First Approach**: Prioritised **responsive design** for seamless viewing across devices.
- **Dark Mode Optimisation**: Leveraged **CSS variables** to support light and dark themes dynamically.

## Outcome

This project strengthened my **UI/UX design skills**, enhanced my **understanding of responsive web development**, and provided a structured platform to document my **professional journey and projects**. It also reinforced my ability to integrate **clean code, best practices, and Agile methodologies** into a functional web application.
